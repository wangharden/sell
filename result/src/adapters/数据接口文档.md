# 需求：

## 市场信息
- 实时快照 get_snapshot(symbol) 
    - 返回买一价（bid_price1）、卖一价（ask_price1）、卖二量（ask_volume2）
    - 对应功能：检查盘口价格、计算中间价。

- 涨跌停价：get_limits(symbol) 
    - 返回一对double（涨停价, 跌停价）。
    - 操作手册中提到要从代码表（TDF_CODE_RESULT）或快照数据算。

- 集合竞价数据：get_auction_data(symbol, date_str, time_str)
    - 返回开盘价和成交金额（jjamt）。
    - 操作手册中是使用tick数据（MSG_DATA_TRANSACTION）过滤时间≤092700的最后一条。




##交易接口
- 持仓查询：query_positions()
    - 返回std::vector<Position>（每个有symbol, total, available）。 （）
订单操作：place_order(req)（下单，返回order_id）、cancel_order(order_id)（撤单）、query_orders()（返回std::vector<Order>）。
其他：连接函数connect_market(host, port, user, pwd)。订阅股票（从CSV加载，像你的g_stockMap）。
